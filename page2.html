03 操作方法

Game

ボタンを押すことで各ウィンドウを展開することが可能です。
今回は使用頻度が多いMotion・Faceウィンドウについて見てみましょう。
※Parameters・Axisウィンドウはほとんど使用しません。

※DefaultPoseは各Motionと各Controllerのみが初期位置に戻ります。Facialsは初期状態に戻りません。
  また、「姿勢保持」をオンにしているAngle Motionを再生し、DefaultPoseを行っても、
  その部位は初期位置に戻らなくなります。

Scene

Scene画面を使用し、以下のように両手の位置や角度を可動域内で自由に動かすことができます。
※操作前に、Debug Settingsで「Motion Edit Mode On」を押す必要があります。
